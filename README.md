# 🤖 AI Code Helper

An AI-powered Python CLI tool that reviews, explains, 
and suggests improvements for your code.

## Features
- 🔍 **Review** — Finds bugs in your code
- 💡 **Explain** — Explains what your code does
- ✅ **Improve** — Suggests better ways to write code

## How to Use
```bash
python main.py review yourfile.py
python main.py explain yourfile.py
python main.py improve yourfile.py
```

## Built With
- Python 🐍
- Groq API
